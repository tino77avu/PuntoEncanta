# Carpeta de Imágenes

## Logo

Coloca tu archivo de logo en esta carpeta con el nombre: **`Logo.png`**

### Formatos aceptados:
- PNG (recomendado con fondo transparente)
- JPG/JPEG
- SVG

### Tamaño recomendado:
- Altura: 50-60px para mejor visualización
- Ancho: proporcional al logo
- Resolución: 2x o 3x para pantallas de alta densidad (Retina)

### Ubicación del archivo:
```
PuntoEncanta/src/main/resources/static/images/Logo.png
```

### Nota:
Si tu logo tiene un nombre diferente o formato diferente, actualiza la referencia en:
- `src/main/resources/templates/index.html` (línea 21)
- Cambia `Logo.png` por el nombre de tu archivo
